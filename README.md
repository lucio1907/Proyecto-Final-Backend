## Run Locally ✅

Clone the project

```bash
  git clone https://github.com/lucio1907/Proyecto-Final-Backend
```

Go to the project directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
